Declara un vector

Instrucción

Descripción

Declara un vector. El tamaño del vector se obtiene a partir del número de elementos del vector constante o de la variable (tipo vector) para realizar una copia del mismo tipo y tamaño.

Campos

  • Nombre: Nombre del vector.

Entradas

  • Valor: Valor a asignar al vector.

Ejemplo

Este ejemplo muestra cómo usar variables (tipo vector). Primero declara una variable (tipo vector) global y local y luego crea otra variable (tipo vector) local. Los elementos de la nueva variable local son asignados con valores de las variables (tipo vector) global y local anteriores. Los valores resultantes se muestran en consola.

Se perderán todos los cambios!
