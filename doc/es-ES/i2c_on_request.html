Enviar Al Demandar I2C

Instrucción

Descripción

Esta instrucción debe ser usada por un esclavo I2C para enviar datos a un maestro I2C (el maestro demanda información a enviar en la dirección indicada). La instrucción ejecuta el código dentro de la sección 'hacer' y devuelve los datos requeridos. Los datos deben estar almacenados en una variable (escalar o vector) o en una constante numérica. Esta instrucción sólo puede utilizarse una vez por programa, ya que si se dispone de múltiples copias de esta instrucción, sólo se renderizará la última de todas.

Campos

  • Dirección: Dirección del esclavo I2C. El valor máximo de la dirección es 127.

Códig apilable

  • Hacer: Código a ejecutar.

Entradas

  • Enviar datos: Datos a enviar (debe ser una variable o una constante numérica).

Ejemplo

Este ejemplo muestra las capacidadades de la comunicación I2C. Representa una red distribuida de MCUs. Una de ellas actua como maestro que solicita información de la lectura de un sensor de infrarrojos de dos esclavos independientes (en las direcciones 20 y 21). Los datos recibidos se adaptan a la escala de 0 a 180 para ser transmitidos a un esclavo que realiza las funciones de controlador de servomotores (en la dirección 10) y que es el encargado de mover los servos.
