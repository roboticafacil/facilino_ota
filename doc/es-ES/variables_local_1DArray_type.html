Declara un vector con tipo

Instrucción

Descripción

Declara un vector especificando el tipo de datos a almacenar. El tamaño del vector se obtiene a partir del tamaño del vector constante o de la variable (tipo vector) sobre la que realizar una copia del mismo tipo (debe coincidir con el tipo de datos indicado) y tamaño.

Campos

  • Nombre: Nombre del vector.
  • Tipo: Selecciona el tipo de variable 'Text', 'caracter', 'entero', 'entero largo', 'entero corto', 'octeto', 'binario', 'decimal', etc...

Entradas

  • Valor: Valor a asignar al vector.
