Declara un vector global (compartido) con tipo

Instrucción

Descripción

Declara un vector global (compartido) con un tipo especificado. Las variable compartidas deben usarse cuando la variable se usa en interrupciones. El tamaño del vector se toma del tamaño del vector constante o del tamaño de la variable (tipo vector) a copiar (en este caso los tipos de variables deben de coincidir).

Campos

  • Nombre: Nombre de un vector.
  • Tipo: Selecciona el tipo de variable 'Entero', 'Entero largo', 'Octeto', 'Binario', 'Decimal', etc..

Entradas

  • Valor: Valor a asignar al vector.
