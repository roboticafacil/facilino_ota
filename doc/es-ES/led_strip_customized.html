Expresión personalizable para tira de 7 LEDs RGB

Instrucción

Descripción

Genera una expresión personalizable para la tira de 7 LEDs RGB. Usar en combinación de las siguientes instrucciones:

Requisitos

Licencia de Facilino.

Requiere instalar la librería NeoPixel https://github.com/adafruit/Adafruit_NeoPixel

Campos

  • Colores: Selecciona el color de cada LED. Selecciona el color negro para apagar el LED.

Salida

Expresión generada.
