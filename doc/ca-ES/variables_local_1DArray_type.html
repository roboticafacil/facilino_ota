Declara un vector amb tipus

Instrucció

Descripció

Declara un vector especificant el tipus de dades a enmagatzemar. El tamany del vector s'obté a partir del tamany del vector constant o de la variable (tipus vector) sobre la que realitzar una còpia del mateix tipus (deu coincidir amb el tipus de dades indicat) i tamany.

Camps

  • Nom: Nom del vector.
  • Tipus: Selecciona el tipus de variable 'Text', 'caràcter', 'sencer', 'sencer llarg', 'sencer curt', 'octet', 'binari', 'decimal', etc...

Entrades

  • Valor: Valor a assignar al vector.
