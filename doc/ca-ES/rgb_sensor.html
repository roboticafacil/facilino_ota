Detecta color RGB

Instrucció

Descripció

Detecta un color amb el sensor RGB TCS3200. Retorna fins a onze possibles colors: negre (0), blanc (1), roig (2), verd fosc (3), blau fosc (4), groc (5), magenta (6), verd clar (7), blau clar (8), morat (9) o taronja (10). Si el color no està entre els valors listats, aleshores retorna "Sense color (-1)".

Requisits

Llicència de Facilino.

Entrades

  • Pin S0: Pin S0 del sensor TCS3200.
  • Pin S1: Pin S1 del sensor TCS3200.
  • Pin S2: Pin S2 del sensor TCS3200.
  • Pin S3: Pin S3 del sensor TCS3200.
  • Pin OUT: Pin OUT del sensor TCS3200.

Eixida

Color detectat.
