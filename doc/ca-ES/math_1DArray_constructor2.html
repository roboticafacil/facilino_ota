Vector (de dos nombres)

Instrucció

Descripció

Defineixix un vector constant amb dos nombres.

Entrades

  • Element 1: Primer element del vector.
  • Element 2: Segon element del vector.

Eixida

Vector constant.

Exemple

Aquest exemple mostra com usar variables (tipus vector). Primer declara una variable (tipus vector) global i local i després crea un altra variable (tipus vector) local. Els elements de la nova variable local són asignats amb valors de les variables (tipus vector) global i local anteriores. Els valores resultants es muestren en consola.
