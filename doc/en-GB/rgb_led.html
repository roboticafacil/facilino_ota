RGB LED

Instruction

Description

Sets the colour of an RGB led.

Inputs

  • Pin R: Red LED Pin.
  • Pin G: Green LED Pin.
  • Pin B: Blue LED Pin.
  • Colour: Colour to generate

Example

In this example, the colour of an RGB LED is changed to the red, green and blue colours sequentially.

All changes will be lost!
