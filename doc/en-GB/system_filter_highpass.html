High-pass filter

Instruction

Description

This instruction implements a discrete Butterworth first order high-pass filter. If the time difference between the current sampling time and previous sampling time is lower than Ts, then the filter returns the previous output, while if the time difference is larger, the filter updates the output based on current input (and previous inputs and outputs), and, obviously, the cut-off frequency.

Requirements

Facilino's license.

Fields

  • Name: Name of the filter (must be different for every filter of the same type).
  • Ts: Sampling time (in milliseconds).
  • Cut-off freq.: The cutoff frequency in % Wn must be 0% < Wn < 100%, with 100% corresponding to half the sampling frequency Fs=1/Ts, being Ts the sampling time expected for each measurement.

Inputs

  • Input: Input signal.

Output

Filtered signal.
