Configures a BLE Service

Instruction

Description

Creates a BLE service to provide custom characteristics. The type of characteristics depend on your service. It is your task to make them compatible with the BLE standards if you are looking for compatibility with external Apps (see https://www.bluetooth.com/specifications/gatt/services/ for details). This instruction must be used in combination of the following instruction:

Fields

  • Name: Name of the service (as it should be recognised by the client).
  • UUID: Universally Unique Identifier for the service. To create custom UUIDs, you can use https://www.uuidgenerator.net/

Mutator

Create new characteristics by adding them to the container:

Container
Elements

Inputs

  • Characteristic: Characteristic to configure (it nees a custom characteristics instruction with attributes and custom UUID).

Output

Returns the service with the characteristics.
