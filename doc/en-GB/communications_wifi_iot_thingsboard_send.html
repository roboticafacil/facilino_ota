ThingsBoard Send

Instruction

Description

This instruction sends data to Thingsboard. Data can be either a time-series value or an attribute. The difference is that time-series values are registered in your Thingsboard application such that the time when received is associated to the data (i.e. to display a time-series chart), while attributes only retrive the value to show (i.e.: a LED status). Add messages to send by including time-series or attribute elements to the container of this mutable instruction. For each time-series or attribute element, you will need to provide the name you have configured in your widget and the value to send. The setup instruction gives you more details for configuring your IoT project:

Requirements

Facilino's license.

Mutator

Container
Elements

Text fields

  • Time-series or attribute name: Name of the time-series or attribute (as configured in your widget).

Inputs

  • Value: Value to send.

Example

This example uses the ESP8266 (integrated) processor to get values from a knob and a switch from a Thingsboard Dashboard. The relay value read from Thingsboard is sent to the digital pin D2 so it is physically interacting. The knob value read from Thingsboard is simply displayed on the console. Temperature values from a DHT11 sensor are sent back to Thingsboard and displayed accordingly on a chart.
