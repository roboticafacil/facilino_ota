Parse an decimal number on the serial port

Instruction

Description

This instruction parses a decimal number received on the serial port. All characters at the beginning of the text string other than digits 0 through 9, "." or the minus sign are ignored. Parsing ends when an invalid character is read. If no character has been received after a specified time, the function returns 0. The time can be set with the instruction:

Output

Parsed decimal number (float)

Example

This example shows how we can read decimal (floating) numbers through the USB port (console)..
