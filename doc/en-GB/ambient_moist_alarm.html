Moisture sensor alarm

Instruction

Description

Checks the moisture alarm levels of the moisture sensor and, if an alarm is triggered, executes the corresponding code. Alarm levels are automatically set so that they are considered within appropriate humidity values for a soil.

Inputs

  • Pin: Analog Pin to which the sensor is connected to.

Checkbox Fields

  • Warn just once: Activate this checkbox to execute the code inside each of the cases just once. The code will not be executed again unless it changes from one state to another.

Statement Instructions

  • High moisture: Set of instructions to be executed in case of reading a moisture higher than the established high-moisture alarm level.
  • Medium moisture: Set of instructions to be executed in case of reading a moisture below the set high-moisture alarm level and below the low-humidity alarm level.
  • Low moisture: Set of instructions to be executed in case of reading a moisture below the set low-moisture alarm level.
