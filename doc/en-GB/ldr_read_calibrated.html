Reads calibrated LDR values

Instruction

Description

Reads a calibrated value of a LDR sensor. The instruction returns a value between 0% and 100% depending on the values of the minimum and maximum lux set for the sensor. Use in combination with the following instructions:

Inputs

  • PIN: Analog PIN where the LDR is connected to.

Output

Returns the calibrated value (a number between 0 and 100) of the LDR sensor.
