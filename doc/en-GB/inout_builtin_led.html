Turns on / Turns off / Toggles the on-board LED

Instruction

Description

Turns on / Turns off / Toggles the integrated on-board LED (connected to Arduino pin 13).

Inputs

  • Status: Drop-down allows you to select the 'on', 'off' or 'toggle' status of the LED.

Example

Switching on, off and toggling the integrated LED.

All changes will be lost!
