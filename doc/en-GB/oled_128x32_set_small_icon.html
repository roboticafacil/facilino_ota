Set 8x8 Icon (OLED 128x32 I2C)

Instruction

Description

Sets an 8x8 small icon on one of the corners in a OLED 128x32 I2C device. The icon can be placed on the top-left, top-right, bottom-left or bottom-right corner of the display. Use in conjunction of the following instruction:

Requirements

Facilino's License.

This instruction requires Adafruit SSD1306 and Adafruit GFX Libraries installed on Arduino IDE. These libraries are available at: https://github.com/adafruit/Adafruit_SSD1306 and https://github.com/adafruit/Adafruit-GFX-Library and also through the Library Manager:

Dropdown list

  • Corner: Top-left, top-right, bottom-left or bottom-right corner.

Inputs

  • Icon: 8x8 icon to display.
