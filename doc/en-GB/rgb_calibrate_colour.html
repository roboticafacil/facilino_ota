Calibrate a colour (for detection)

Instruction

Description

Calibrates the maximum and minimum values ​​(in milliseconds) detected with the different filters for objects with the same colour (or very similar colour variations). This instruction affects the colour detection instruction:

Requirements

Facilino's License.

Inputs

  • Colour: Colour filter to adjust.
  • Min: Value in milliseconds minimum.
  • Max: Value in milliseconds maximum.

Example

In this example, the word 'RED', 'GREEN' or 'BLUE' is displayed on the console depending on whether the red, green or blue colour is detected using the TCS3200 sensor. The colours have been previously calibrated.

All changes will be lost!
