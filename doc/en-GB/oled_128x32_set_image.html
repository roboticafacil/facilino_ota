Set Image (OLED 128x32 I2C)

Instruction

Description

Draws an image in a OLED 128x32 I2C device. The coordinate indicates the top-left corner of the image. Use in combination with the following instructions:

Requirements

Facilino's License.

This instruction requires Adafruit SSD1306 and Adafruit GFX Libraries installed on Arduino IDE. These libraries are available at: https://github.com/adafruit/Adafruit_SSD1306 and https://github.com/adafruit/Adafruit-GFX-Library and also through the Library Manager:

Inputs

  • Image: Image to draw.
  • X0: Top-left corner X coordinate in pixels.
  • Y0: Top-left corner Y coordinate in pixels.
