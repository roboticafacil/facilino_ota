Linear Mapping (advanced)

Instruction

Description

Linearly transforms (mapping) a value between two numbers "From [min - max]" to another value included between two other numbers "to [min - max]". Allows you to set a value to be within the specified linear setting.

Inputs

  • Value: value to convert.
  • From (min): minimum value of the input range.
  • From (max): maximum value of the input range.
  • to (min): minimum value of the output range.
  • to (max): maximum value of the output range.

Output

Converted value.
