Facilino Setup

What do I need?

Welcome to Facilino setup. This guide will help you setting Facilino.

Requirements:

  • Computer or laptop.
  • Arduino IDE or Visual Studio Code with PlatformIO IDE installed (see installation instructions).
  • Any Arduino-based electronics (see 'Hardware settings' below).

Icon Menu

File

Here you will find a sub-menu with tools for creating a new program, upload an exisiting Facilino program, download a Facilino file with your current code or even to download the required, export your Facilino code to an ino file, firmware for the first-time use of the electronics.

Edit

In this sub-menu, you have 'Undo' and 'Redo' tools.

Language settings

Facilino has been translated into many languages, please feel free to choose the language that suits your needs by clicking on the 'Language' icon in the top menu . Once selected, Facilino page will be reloaded.

Officially, English and Spanish are supported, but many other languages are fully operational (or mostly fully operational). Please, take into account that some translations may not contain a specific translated block instruction or help document. If you want to contribute to the translation of this software, please, contanct Leopoldo Armesto at soporte@roboticafacil.es.

Facilino version

There are four versions: Basic Facilino, Advanced Facilino, Basic Facilino OTA and Advanced Facilino OTA.

  • Basic Facilino: It allows you generate Arduino code using simple block instructions. The kind of things you can do with the Basic version are obviously limited compared to the Advanced one, but still it allows you to develop amazing projects with simple block instructions. See migrating from Basic Facilino to Advanced Facilino section to learn how to develop advanced code and continue learning coding Arduino electronics.
  • Advanced Facilino: Just as Basic Facilino, but with an extended block instruction set and some extended options within the block instructions.
  • Basic Facilino OTA: Just as Basic Facilino, but it allows you to upload code directly from your tablet or iPAD using OTA (Over-The-Air) thechnology. It requires an additional computer, teacher's computer, facility server or even your own computer to upload the code to compile. Please, follow the setting instructions to learn how to set Facilino OTA (see the instructions below).
  • Advanced Facilino OTA: Just as Advanced Facilino, but it allows you to upload code directly from your tablet or iPAD using OTA (Over-The-Air) thechnology. It requires an additional computer, teacher's computer, facility server or even your own computer to upload the code to compile. Please, follow the setting instructions to learn how to set Facilino OTA (see the instructions below).

Hardware settings

Facilino instructions and code generation are adapted to the selected hardware (electronics). This represents a great advantage over classic text programming environments such as Arduino IDE, because the same block can be used with different hardware settings, which helps you to focus on the most important aspects of coding, rather than focusing on details. If you want to use a different processor in future projects, programming with Facilino will likely imply very few changes in your code. For that reason, selecting of your hardware is one of the first things you will need to do when programming with Facilino. Please, select your correct hardware setting by clicking on the 'Hardware settings' icon in the top menu . Once selected, Facilino page will be reloaded.

Uploading code

This version of Facilino cannot upload code directly. It just generates the code that can be uploaded using a different tool such as classic Arduino IDE or a new IDE known as PlatformIO. We highly encourage to use PlatformIO, instead of classic Arduino IDE, because the PlatformIO Library Manager solves many library dependencies that can cause compilation errors. Here you have links to downloading papges:

  • Arduino IDE
  • PlatformIO (requires Visual Studio Code)

Please, follow installation instructions with the links below.

First thing you need to do to upload your code is to export your code to an Arduino IDE file, so once your done with Facilino coding, export the code in the File menu and save the 'ino' file in a folder under the same name.

If you use Arduino IDE, you need to open the file using Arduino IDE and select your processor, port to be able to upload your code (this step depends on the hardware setting you are using). For instance, ESP board such as ESP8266 and ESP32 need additional setup instructions in order to be able to compile and upload code (see links below). Some block instructions require additional libraries. By default Arduino IDE come with some libraries installed, but others need to be installed manually. Use the 'Verify' and 'Upload' icons in the top-left side to proceed uploading your code (you don't need to verify your code every time, you can directly use the upload button).

On the other hand, PlatformIO is integrated in Visual Studio Code, so if you want to use PlatformIO, you need to run Visual Studio Code and click on PlatformIO icon on the left side icon bar. Then, open to the PIO Home file and select 'Import Arduino Project'. Select your board (processor) and folder with the 'ino' file. Then, leave the magic behind PlatformIO to do the rest, because all dependencies, libraries and required additional files will be automatically installed. You can 'Build' and 'Upload' your code using the icons at the left-bottom side of the application.

If you want to modify your code, you can go back to Facilino and modify your block instructions and copy your new generated code and paste it into your exisiting Arduino code (no need to export the file 'ino' file again, just copy the code using the utility icon in Facilino in the icon menu bar and paste it in Arduino IDE) .
