RGB LEDs strip

Instruction

Description

Sets the colors of a RGB LEDs strip with a given number of pixels. To turn off the light of a LED, we must select the black colour.

Requirements

It requires to install NeoPixel library https://github.com/adafruit/Adafruit_NeoPixel

Inputs

  • PIN: 1-wire digital interface for data.
  • Pixels: Number of pixels.

Fields

  • Colours: Colour to display on each of the pixels
