Atmospheric pressure barometer (BMP180)

Instruction

Description

Returns the atmospheric pressure of the barometric sensor BMP180 in Pascals (the sensor is connected to the I2C bus).

Requirements

Facilino's License.

This instruction requires BMP085 Adafruit Library installed on Arduino IDE. This library is available at: https://github.com/adafruit/Adafruit-BMP085-Library and also through the Library Manager:

Output

Atmospheric pressure in Pascals.

Example

This example shows the barometric pressure, temperature, and estimated altitude measurements with the barometric sensor BMP180.

All changes will be lost!
