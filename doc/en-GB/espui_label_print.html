Modifies label text (web-based user-interface for ESP)

Instruction

Description

Modifies (updates) the label text of an named label on a web-based user interface for ESP processors

Combine this instruction with the following instruction:

Inputs

  • Label: Label name to update.
  • Text: New text to display.

Example 1

This example shows a simple user interface for remotely monitoring a temperature and humidity read with the DHT11 sensor. The user-interface IP is 192.168.1.1 and the SSID for the hot-spot is "ESP8266_remote" (change accordingly to avoid conflict if others might be using this name).

All changes will be lost!
