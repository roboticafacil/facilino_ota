Robot grip

Instruction

Description

Controls the movements of a robot's grip with independent or coordinated control of the fingers. The opening and closing positions of each finger can be set with the instruction:

Note that for ESP32 this library is required:

https://github.com/jkb-git/ESP32Servo

Requirements

Facilino's License.

Inputs

  • Left: PWM pin to which the left finger position servo is connected to.
  • Right: PWM pin to which the right finger position servo is connected to.

Dropdown lists

  • Primitive: Grip movement primitive (individual or coordinated opening or closing of the fingers).
