Predefined expression for a 7-RGB LEDs strip

Instruction

Description

Generates a predefined expression for a 7-RGB LEDs strip. Use in combination with:

Requirements

Facilino's license.

It requires to install NeoPixel library https://github.com/adafruit/Adafruit_NeoPixel

Fields

  • Expression: Predefined expression to generate.
  • Colour: Colour of the expression (all LEDs have the same colour). Select black to turn OFF all LEDs

Output

Generated expression.

Example

In this example, we show several examples to generate expressions with a 7-RGB LEDs strip using predefined expressions with different colours.
