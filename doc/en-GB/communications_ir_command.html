Decodes IR data as commands

Instruction

Description

This instruction process IR data and executes a command (a code) if received.

Inputs

  • PIN: Digital PIN where the IR receiver is connected to.

Mutator

Container

Add codes (commands) to decode.

Code

Write the code to decode and include the set of instructions to execute in the statement input 'Do'.

  • Do: Set of instructions to execute in case of IR reciever has received any data.
