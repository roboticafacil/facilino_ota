Rain level alarm

Instruction

Description

Discriminates the amount of rain from the rain sensor and executes the corresponding code. Alarm levels can be set with the instruction:

Requirements

Facilino's License.

Inputs

  • Pin: Analog pin to which the sensor is connected to.

Statement Instructions

  • Intense rain: Set of instructions to run in case of rain (intense).
  • Soft rain: Set of instructions to run in case of rain (soft).
  • Dry: Set of instructions to run in case there is no rain.
