Generates predefined expression in LEDs matrix

Instruction

Description

It allows you to generate a predefined expression for the robot DYOR with the LEDs matrix (max7219). It allows to select between five predefined expressions for the LEDs matrix. Use in combination with:

Requirements

Facilino's License.

Dropdown lists

  • Expression: Angry, Happy, Sad, In Love, Surprised.

Output

Predefined expression

Example

In this example, the DYOR robot generates the expression 'Happy' and 'Angry' alternatively with waiting times of 1000 milliseconds.

All changes will be lost!
