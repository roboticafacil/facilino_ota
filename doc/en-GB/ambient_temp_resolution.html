Sensor Temperature Resolution (DS18B20)

Instruction

Description

Sets the resolution of the DS18B20 temperature sensor, with higher resolutions, the reading is more accurate.

Requirements

Facilino's License.

This instruction requires Dallas Temperature installed on Arduino IDE. This library is available at: https://github.com/milesburton/Arduino-Temperature-Control-Library and also through the Library Manager:

Dropdown lists

  • Bits: Sensor resolution bits.

Inputs

  • Bus Pin: Pin to which the sensor is connected to.

Example

In this example the temperature read with the DS18B20 sensor at different resolutions are displayed on the console.

All changes will be lost!
