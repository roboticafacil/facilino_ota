Detects RGB color with sensor TCS3200

Instruction

Description

Detects a colour with the RGB sensor TCS3200. Returns up to eleven possible colours: black (0), white (1), red (2), dark green (3), dark blue (4), yellow (5), magenta (6), light green (7), light blue (8), purple (9) or orange (10). If sensed colour is not within the listed values, then returns "No Colour (-1)".

Requirements

Facilino's License.

Inputs

  • Pin S0: Pin S0 of sensor TCS3200.
  • Pin S1: Pin S1 of the TCS3200 sensor.
  • Pin S2: Pin S2 of the TCS3200 sensor.
  • Pin S3: Pin S3 of TCS3200 sensor.
  • Pin OUT: Pin OUT of TCS3200 sensor.

Output

Color detected.

Example

In this example, the colour names are displayed on the console if detected using the TCS3200 sensor.
