Set opening and closing positions of the robot's grip

Instruction

Description

Sets the opening and closing position of the grip.

Requirements

Facilino's License.

Inputs

  • Degrees: Position in degrees to be adjusted.

Dropdown lists

  • Finger and position: Determines the finger (left or right) and mode to adjust (open or close).

Example

In this example, the opening and closing angular positions of grip fingers are set and a demonstration movement of the grip is then performed.

All changes will be lost!
