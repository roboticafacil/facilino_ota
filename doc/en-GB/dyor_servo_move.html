Position Servo

Instruction

Description

Generates a PPM signal for controlling a position servo (SG90 and similar ones).

Note that for ESP32 this library is required:

https://github.com/jkb-git/ESP32Servo

Inputs

  • Pin: PWM pin to which the position servo is connected to.
  • Angle: Angle of rotation (between 0º and 180º).

Example

This example shows how to control the position of a servo by performing a progressive scan from 0 ° to 180 ° and then rapidly moving it to 0 ° again.

All changes will be lost!
