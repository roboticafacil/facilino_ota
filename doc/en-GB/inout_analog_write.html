Write Analog Pin

Instruction

Description

Write an analog value on a digital pin (with dual PWM function) with an approximate frequency of 490Hz (pins 5 and 6 have an approximate frequency of 980Hz on Arduino Uno boards and similar) on Arduino boards and NodeMCU or user-defined in ESP32 processors. The input value (the duty cycle) is between 0 and 255 for Arduino boards because they use a 8-bit resolution, while is ESP32 processor the maximum value depends on the resolution with a maximum value of 2^resolution_bits-1. The pin can be indicated with the instruction:

Inputs

  • Digital Pin (PWM): A digital pin on which to generate the PWM signal.
  • Dutycycle: value of the analog signal to be generated (between 0 and 2^resolution_bits-1).
In ESP32 processors we can select the PWM frequency as well as the resolution (channel is automatically selected based on the pin number). The maximum frequency is 40Mhz for 1 bit resolution, this frequency decreases with the selected resolution so that maximum frequency is 80000000/(2^resolution_bits), i.e.: for a resolution of 8 bits, the maximum frequency is 312.5Khz.

Fields

  • Frequency: Frequency in Hz for the PWM signal.
  • Resolution: Number of resolution bits .
