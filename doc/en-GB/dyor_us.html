Get distance with ultrasonic sensor (HC-SR04)

Instruction

Description

Gets the distance in centimeters from the ultrasonic sensor HC-SR04. Between measurements we must ensure that at least 60ms time gap.

Additional note

The instruction uses a variant of the 'pulseIn' instruction of Arduino that allows it to be used within interrupt 0 (multi-task instruction).

Inputs

  • Eco: Pin of the echo signal (ECHO).
  • Trigger: Pin of the trigger signal (TRIGGER).

Output

Returns the distance in centimeters.

Example

This example shows the distance information read with the ultrasonic sensor via the serial port (console).
