Combines two 16-bits numbers into a 32-bit number

Instruction

Description

This instruction combines two 16-bits numbers into a 32-bit number by assigning the 16-bit most significat bits to the first number and the 16-bit least significant bits to the second number.

Inputs

  • Numbers: The first number is assigned to the most signficant bits and the second number to the least significant bits.

Output

Returns a 32-bit number with the resulting operation.
