Interrupt counts

Instruction

Description

Returns the number of times a signal has triggered an interrupt. Interrupts can be set in "low state", "change", "rising edge" or "falling edge" mode. This instruction is expected to be included in main loop code, while the code implemented for the interrupt is simply increasing a counter.
