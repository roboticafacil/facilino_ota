Facilino is a web-based application to generate code and program Arduino-based electronics (several processors, such as Arduino Uno, Arduino Nano, ESP8266, ESP32, etc... can be used). Facilino is intended to be used in classroom/labs by young learners and anyone at home for non-professional code developing. There are three versions: Facilino Junior, Facilino, and Facilino OTA.

  • Facilino Junior: It allows you generate Arduino code using simple block instructions. The kind of things you can do with the Junior version are obviously limited compared to the Advanced one, but still it allows you to develop amazing projects with simple block instructions. See migrating from Basic Facilino to Advanced Facilino section to learn how to develop advanced code and continue learning coding Arduino electronics. It needs Arduino IDE or PlatformIO to compile and upload code (see setup instructions below).
  • Advanced Facilino: Just as Facilino Junior, but with an extended block instruction set and some extended options within the block instructions. It needs Arduino IDE or PlatformIO to compile and upload code (see setup instructions below).
  • Facilino OTA: Just as Facilino, but it allows you to upload code directly from your tablet or iPAD using OTA (Over-The-Air) thechnology. It requires an additional computer, teacher's computer, facility server or even your own computer to upload the code to compile. Please, follow the setting instructions to learn how to set Facilino OTA (see setup instructions below).

Access the help of each instruction (on the left) to show the documentation and usage examples. Documentation adapts to your language settings, being English and Spanish, the official ones. If the documentation is not available in your native language, then the English version will show up. Documentation also adapts to your Facilino version as well as to your hardware settings.
