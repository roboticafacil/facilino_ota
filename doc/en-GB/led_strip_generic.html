7-RGB LEDs strip

Instruction

Description

Sets an 'expression' on a 7-RGB LEDs strip. Use in combination with:

Requirements

Facilino's license.

It requires to install NeoPixel library https://github.com/adafruit/Adafruit_NeoPixel

Inputs

  • PIN: 1-wire digital interface for data.
  • Expression: An 'expression' to set ON or OFF the LEDs.
