Gets characteristic data

Instruction

Description

If a characteristic checks the 'Write' attribute, then this instruction returns the associated data for that characteristic as an array of bytes (use the length' instruction to know the array length). It's contents depends on the characteristic and it is application dependant. This instruction must be used in combination of the following instructions:

Dropdown Lists

  • Characteristic: Name of the characteristic to use.

Output

Returns an array with the characteristic data.
