Discriminates the status of a button

Instruction

Description

Discriminates the state of a button and executes a code in case it is pressed and another code in case it is not pressed.

Inputs

  • Pin: Pin (digital input) to which the button is connected to.

Statement Instructions

  • Pushed: Instructions to be executed in case of pressing the button.
  • Not pushed: Instructions to execute in case of NO push button.

Example

Shows a message on the console when pressing the button.

All changes will be lost!
