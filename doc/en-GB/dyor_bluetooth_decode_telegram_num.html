Decode telegram

Instruction

Description

Returns the value of the telegram data in the indicated position (starts with 0). Use in conjunction with instruction:

Requirements

Facilino's License

Inputs

  • Position: Telegram data position.

Output

Decoded data.
Structure telegram
STXCMDLengthData 0...Data N-1END

STX being the '@' symbol in ASCII; CMD command (1 byte); Length the length of the data; Data 0 ... Data N-1 data and END the '*' symbol in ASCII.
