Draws a stream of text or image in the LEDs matrix

Instruction

Description

This instruction accepts a stream (corresponding to a stream of text or image) to be shown in the 8x8 LEDs matrix (max7219). Given a delay time, the text or image will be shifted to the left (one column at each step) to show to complete text/image stream. Text and image shift can be done smoothly (one column each shift or individually). The instruction can be executed in the main flow such that it waits until the animation is over to execute next instruction or it can be used in the TIMER 0 interrupt as a background task. In the latter case, the next instruction will be executed immediately such that the main flow does not wait until the animation is over. Use in combination with the following instructions:

Requirements

Facilino's License.

Inputs

  • CS: Pin CS of the LED matrix (max7219).
  • DIN: Pin DIN of the matrix of LEDs (max7219).
  • CLK: Pin CLK of the LED matrix (max7219).
  • Orientation: Orientation of the LED matrix (max7219).
  • LED Matrix Stream: Text or LED Matrix 'images' to show in a stream.
  • Wait: Transition wait time between consecutive streams.
  • Smooth shift?: If selected, the text or image is shifted by columns (one column once time have elapsed). Otherwise, the shift is done in 8 by 8 columns so each letter/image appears individually.
  • Background task?: If selected, the LED matrix animation starts and next instruction starts immediately (it doesn't wait until the animation is complete). Otherwise, the animation is executed and waits until it finishes to execute next instruction.

Dropdown list

  • Configuration: Configuration of the LEDs matrix (vertical, horizontal, vertical 180º or horizontal 180º).
