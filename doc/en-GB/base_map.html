Linear Mapping

Instruction

Description

Linear scaling (mapping) of a number between the range [0 - 1023] in another range between [0 - 'range']. It allows you to set a value (usually of the analog inputs) in another range, eg from 0 to 255.

Inputs

  • Value: value to convert.
  • Max: maximum value of the output range.

Output

Converted value.
