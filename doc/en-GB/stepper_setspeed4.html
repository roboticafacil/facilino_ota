Sets rotation speed (for a stepper motor)

Instruction

Description

This instruction is based on Arduino Stepper library, which allows to define the speed of a stepper motor in RPM.

Fields

  • Total steps: Number of steps for the stepper motor to complete a turn (i.e.: for 28BYJ-48 motor this corresponds to 2048 steps).

Dropdown lists

  • PIN1 to PIN4: Pins to control the motor with 4 pins (might depend on the driver/motor configuration (for ULN2003A and 28BYJ-48 corresponds to IN1, IN3, IN2 and IN4, respectively)

Inputs

  • Set Speed (RMP): Rotation speed in RPM (maximum tested speed with ULN2003A driver and stepper motor 28BYJ-48 is 17 RPM).

Example

In this example, a single stepper motor is moved counter-clockwise and clockwise 2048 steps (a whole turn) and then stops for 2 secs on every turn.

All changes will be lost!
