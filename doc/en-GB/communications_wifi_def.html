Defines WiFi ESP8266

Instruction

Description

Defines the WiFi for the module ESP8266 or ESP processors.

Requirements

For the ESP8266 module, the instruction requires the ITEADLIB_Arduino_WeeESP8266 library to be installed https://github.com/itead/ITEADLIB_Arduino_WeeESP8266

For ESP processors, the instruction requires having configured external URLs for the board management in the preferences of Arduino IDE:

Subsequently, the ESP8266 card must be installed on the Arduino IDE Board Manager:

Inputs

  • SSID: Network Name (SSID).
  • Password: Password to access the network.
  • Speed: Baud rate (maximum 38400bps) [only for ESP8266 module].
  • TX: Arduino transmission pin (connect to RX of ESP8266 module) [only for ESP8266 module].
  • RX: Arduino receive pin (connect to TX of ESP8266 module) [only for ESP8266 module].

Example

This example uses the ESP8266 WiFi module with Arduino UNO. In the example the first configuration steps are made for the connection to the network and then in the main loop a text message is sent to a server at the IP address 192.168.1.100 through the port 49152 (Arduino is the client of the connection ). If a response is received from the server, the message is displayed on the console via the USB serial port.
