Predefined mouth expression (7-RGB LEDs strip)

Instruction

Description

Generates a predefined expression for the DYOR robots with a 7-RGB LEDs strip (ws2812).

Requirements

Facilino's License.

It requires to install NeoPixel library https://github.com/adafruit/Adafruit_NeoPixel

Inputs

  • PIN: 1-wire digital interface for data.
  • Expression: Expression to show.
  • Colour: Colour to display

Example

In this example, we shown several expressions using the 7-RGB LEDs strip with Arduino Nano
