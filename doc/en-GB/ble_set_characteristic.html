Sets characteristic data

Instruction

Description

This instruction allows to update the characteristic value (data) associated to it, so that it will return the information when requested if the 'Read' attribute has been checked. It's contents depends on the characteristic and it is application dependant. This instruction must be used in combination of the following instructions:

Dropdown Lists

  • Characteristic: Name of the characteristic to use.

Inputs

  • Data: data array or variable with the new characteristic value.
