ESP Deep Sleep

Instruction

Description

Sends to a deep sleep state and wake up after a given amount of time to save energy. In order for this instruction to work properly, the RST pin must be connected to D0 for ESP8266 boards (ESP32 does not require such connection)

Requirements

The ESP8266 card must be installed on the Arduino IDE Board Manager:

For ESP processors, the instruction requires to the latest version of the ESP32 software:

https://github.com/espressif/arduino-esp32

and to follow installation instructions:

https://github.com/espressif/arduino-esp32#installation-instructions

Input

  • Time: Waiting time in seconds.
