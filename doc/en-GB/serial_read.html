Read a byte through serial port

Instruction

Description

This statement reads a byte through the serial port (returns an 'int'). If no data is available, returns -1.

Output

First byte received through the serial port.

Example

This example waits for the letter 's' to begin with the rest of the program. In the ASCII table, the letter 's' corresponds to the value 115.

All changes will be lost!
