Remote Value ThingsBoard (RPC)

Instruction

Description

Returns the variable of a Remote Procedure Call (RPC) of a switch or knob in a Thingsboard project. This instruction can only be used within the context of the corresponding RCP instructions to execute. Associated instruccion:

Requirements

Facilino's license.

Output

Returns either a Boolean (for a remote switch) or a Float (for a remote knob) with the RPC value.

Example

This example uses the ESP8266 (integrated) processor to get values from a knob and a switch from a Thingsboard Dashboard. The relay value read from Thingsboard is sent to the digital pin D2 so it is physically interacting. The knob value read from Thingsboard is simply displayed on the console. Temperature values from a DHT11 sensor are sent back to Thingsboard and displayed accordingly on a chart.

All changes will be lost!
