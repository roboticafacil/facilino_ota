Humidity sensor alarm (DHT11/DHT21/DHT22)

Instruction

Description

Checks the humidity alarm levels of the DHT11/DHT21/DHT22 sensor and, if an alarm is triggered, executes the corresponding code. Alarm levels are automatically set so that they are considered within the comfort humidity of a room and uses both temperature and humidity to determine the appropriate situation:

Requirements

This instruction requires DHT Adafruit Library and Adafruit Unified Sensor Library installed on Arduino IDE. These libraries are available at: https://github.com/adafruit/DHT-sensor-library and https://github.com/adafruit/Adafruit_Sensor and also through the Library Manager:

Inputs

  • Bus Pin: Pin to which the sensor is connected to.

Checkbox Fields

  • Warn just once: Activate this checkbox to execute the code inside each of the cases just once. The code will not be executed again unless it changes from one state to another.

Statement Instructions

  • High humidity: Set of instructions to be executed in case of reading a humidity higher than the established high-humidity alarm level.
  • Medium humidity: Set of instructions to be executed in case of reading a humidity below the set high-humidity alarm level and below the low-humidity alarm level.
  • Low humidity: Set of instructions to be executed in case of reading a humidity below the set low-humidity alarm level.
