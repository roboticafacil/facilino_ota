Every elapsed time

Instruction

Description

Defines a set of 'alternative' cases to do. It does only one case each time the instruction is executed (when the last case is executed, the first case will be executed next time). This instruction is intended to be used within Arduino's loop.

Mutator

Create new cases by adding them to the container:

Container
Elements

Statement Instructions

  • Do: Set of instructions to do to be executed for that case.
