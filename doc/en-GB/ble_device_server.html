Configures a BLE Device Server

Instruction

Description

Creates a BLE device to provide custom services. The type of services depend on your device. It is your task to make them compatible with the BLE standards if you are looking for compatibility with external Apps (see https://www.bluetooth.com/specifications/gatt/services/ for details). This instruction must be outside the Setup/Loop instruction and can be included just once on each program. You can detect when a new client is connected or disconnected and add the corresponding code to those events. To configure a new service, use the following instruction:

Fields

  • Name: Name of the device (as it should be recognised by the client).

Statement Instructions

  • On Connect: set of instructions that are executed when a client is connected.
  • On Disconnect: set of instructions that are executed when a client is disconnected.

Mutator

Create new services by adding them to the container:

Container
Elements

Inputs

  • Service: Service to configure (it nees a custom service instruction with characteristics).
