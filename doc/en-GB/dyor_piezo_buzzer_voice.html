Buzzer (voice)

Instruction

Description

Emits a voice sound with a buzzer. This instruction only works by connecting the buzzer to the Arduino pin 3 to generate a PWM signal. The quality of the buzzer can clearly affect the voice, which can be heard clearly digitized and of low quality.

Requirements

Facilino's License.

Inputs

  • Voice: Pointer to a variable that contains a word.

Example

This example shows how to generate a phrase with the words "DANGER DANGER MOTOR IS ON FIRE".
