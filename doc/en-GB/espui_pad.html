Creates a button pad (web-based user-interface for ESP)

Instruction

Description

Creates a button pad on a web-based user interface for ESP processors such that it allows you to set its name, colour, initial state and code to execute for different options:

Inputs

  • Pad: Pad name (unique name).
  • Colour: Pad colour.
  • Center button?: Select 'Yes' to include a center button.

Stackable instructions

Each of the cases of the button pad (left, right, top, bottom, center), either pressed or released.
