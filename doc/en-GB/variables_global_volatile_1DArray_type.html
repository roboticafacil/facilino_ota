Declares a global (shared) array with type

Instruction

Description

Declares a global (shared) array using an array constructor specifying its type. Shared variables should be used when a variable is used in interrupts. The size of the array is taken from the size of the array constructor. It can also copy the values of an existing array variable, taking the same size and variable type as the input variable.

Fields

  • Name: Array name.
  • Type: Select variable type 'interger', 'long integer', 'byte', 'boolean' or 'float'.

Inputs

  • Value: Value to assign to the array.
