Setup an Amazon Echo devices

Instruction

Description

This instruction allows you to create digital or analog devices to be connected to Amazon Echo 2.

Mutator

Add as many devices (digital or analog) as desired and respond to the requested order consequently.

Container
Digital device

Set the device name (must be unique) and get the requested state for that device (HIGH or LOW).

Analog device

Set the device name (must be unique) and get the requested state for that device (a value between 0 and 255, so if you say to Alexa to set it to 50%, then you will receive 128).
