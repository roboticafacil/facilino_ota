Creates a block section

Instruction

Description

This instruction adds a comment on the generated code as well as encansulates the code inside the container (generated code include brackets that might be usefull to create different contexts for variables). Also, if you collapse the block instruction, will appear just the comment and the remainder of blocks will be occluded, but it does not affect to the code generation, and thus it is usefull to generate compact Facilino code.
