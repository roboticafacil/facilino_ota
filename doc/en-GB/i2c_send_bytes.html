Send bytes I2C

Instruction

Description

This instruction is used by I2C Master to send data to a I2C slave at the indicated address. The instruction uses the provided variable to determine the total amount of bytes to send (depending on the number of elements if it is an array variable and variable type). The size of 'byte' and 'char' is 1, the size of 'short int' is 2, the size of 'int' is 2, the size of 'long int' is 4, the size of 'float' is 4.

Inputs

  • Variable: Array variable with data to send.

Fields

  • Address: Address of I2C slave. Maximum address value is 127.

Example

This example shows the capabilities of the I2C communication. It represents a distributed network of MCUs. One of them is the Master that requests infrared readings of two independent I2C slaves (addresses 20 and 21). Received data is scaled into a servo position value (upto 180º) and transmitted to a servo controller slave (address 10) which is in charge of moving the servos.
