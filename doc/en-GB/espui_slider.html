Creates a slider (web-based user-interface for ESP)

Instruction

Description

Creates a slider on a web-based user interface for ESP processors such that it allows you to set its name, colour and code to execute when the slider changes its value. This instruction must be stacked inside this instruction (the order affects the way the different components are shown in the user-interface):

To retrieve the value of the slider, use this instruction:

Inputs

  • Slider: Slider name (unique name).
  • Colour: Slider colour.

Stackable instructions

  • Code: Instructions to execute when the slider changes its value.

Example

This example shows a simple user interface for remotely control a servo using a slider. The user-interface IP is 192.168.1.1 and the SSID for the hot-spot is "ESP8266_remote" (change accordingly to avoid conflict if others might be using this name).
