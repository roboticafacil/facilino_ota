Logical SHIFT operation

Instruction

Description

This instruction performs a bitwise shift of the given amount of bits on a number. Shifting a number is equivalent to multiply the number by 2^n, being n the number of number of bits to shift.

Inputs

  • First operand: Number to shift.
  • Second operand: Amount of bits to shift.

Output

Returns a number with the resulting shifting operation.
