Read Analog Pin

Instruction

Description

Reads the value of an analog input pin (from A0 to A7). The reading value is between 0 and 1023. The pin can be indicated with the instruction:

Inputs

  • Analog pin: Analog pin value from A0 to A7.

Output

Read analog value (value between 0 and 1023).

Example

This example shows the analog value of pin A0 on console.
