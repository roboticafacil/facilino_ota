Defines a DYOR bPED Robot

Instruction

Description

Defines a DYOR bPED Robot for walking primitives generation.

Requirements

Requires installation of HIPPIE library https://github.com/roboticafacil/HIPPIE

Facilino license

Inputs

  • Leg 1: Pin for right leg servo.
  • Leg 2: Pin for left leg servo.
  • Ankle 1: Pin for right ankle servo.
  • Ankle 2: Pin for left ankle servo.

Fields (Settings)

  • Leg 1: Angle offset of right leg servo for adjustment in degrees.
  • Leg 2: Angle offset of left leg servo for adjustment in degrees.
  • Ankle 1: Angle offset of right ankle servo for adjustment in degrees.
  • Ankle 2: Angle offset of left ankle servo for adjustment in degrees.

Example

This example configures DYOR bPED robot to be standing at its home position (all servos at 90ยบ).

All changes will be lost!
