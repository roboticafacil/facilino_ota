Scroll a text on a LCD (2x16)

Instruction

Description

This instruction scrolls a text on a LCD screen. You can indicate the delay on each character for scrolling the text and the row to perform the scroll.

Requirements

Requires installing the 'LiquidCrystal I2C' library in Arduino IDE.

Inputs

  • Text: Text to scroll.
  • Delay: Time to delay between each character of the text.
  • Row: Select the row to print the text (0 or 1).
