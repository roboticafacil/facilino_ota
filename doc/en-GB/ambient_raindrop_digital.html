Reads digital rain sensor

Instruction

Description

Returns the value of the rain sensor digital signal. The sensitivity of the potentiometer must be adjusted to determine if it is raining or not.

Inputs

  • Pin: Digital pin to which the sensor is connected to.

Output

True or false depending on whether it rains or not.

Example

In this example, the analogue reading of the rain sensor is shown on the console and in case its digital output is activated because it rains, shows a rain warning message.

All changes will be lost!
