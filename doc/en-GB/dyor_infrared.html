Infrared Sensor (TCRT5000)

Instruction

Description

Gets the amount of light received by the TCRT5000 infrared sensor using the analog output (AO).

Inputs

  • Pin: Pin of the analog signal (AO) of the TCRT5000 sensor.

Output

Returns the analog value of the amount of light (close to 0 with pure white and close to 1023 for pure black).

Example

This example shows the information read with the infrared sensor TCRT5000 via the serial port (console).
