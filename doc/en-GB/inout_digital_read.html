Read digital pin

Instruction

Description

Reads the value of a digital input pin (from 0 to 13, although analog pins can also be used for the same purpose). The reading value is 'HIGH' or 'LOW'. The pin number can be indicated by the instruction

Inputs

  • Pin: Digital pin to read.

Output

Digital value read (HIGH or LOW).

Example

In this example, the integrated LED (pin 13) lights up if the buttons are pressed at the same time or if no button is pressed. If only one of the buttons is pressed, the LED turns off.

All changes will be lost!
