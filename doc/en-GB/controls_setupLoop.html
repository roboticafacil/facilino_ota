Setup / Loop

Instruction

Description

This instruction must be in every Arduino program and in fact is included by default when starting a new code. It allows to include instructions that will be executed at startup (only during startup) and instructions that will be repeated indefinitely (at the end of the last instruction it starts to execute the first instruction again).

Statement Instructions

  • Setup: set of instructions that are only executed once (at startup).
  • Loop: set of instructions that are executed repeatedly.
