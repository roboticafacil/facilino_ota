Returns the value of a slider (web interface for ESP)

Instruction

Description

Returns the actual value of a slider. Use only within the context of the code to stack of this instruction:

Output

Slider value (integer between 0 and 100).

Example

This example shows a simple user interface for remotely control a servo using a slider. The user-interface IP is 192.168.1.1 and the SSID for the hot-spot is "ESP8266_remote" (change accordingly to avoid conflict if others might be using this name).
