Configures encoders

Instruction

Description

Configures the encoders of a mobile robot base.

Requirements

Facilino's license.

Parameter Fields

  • Counts/Turn: Number of counts per turn of the encoders.
  • Wheel diameter: Wheel diameter in cm.
  • Wheel separation: Distance between wheels in cm.

Dropdown lists

  • Left encoder: Interrupt pin where to the left encoder is connected.
  • Right encoder: Interrupt pin where to the right encoder is connected.

Example

In this example, the wheel speeds, left and right wheels, of a mobile base are measured.

All changes will be lost!
