Configures a BLE Characteristic

Instruction

Description

Creates a custom BLE characteristic based on the BLE2902 characteristic descriptor. The characteristic can implement the 'read', 'write' or 'notify' attributes or a combination of them. This instruction must be used in combination of the following instructions:

Fields

  • Name: Name of the characteristic (as it should be recognised by the client).
  • UUID: Universally Unique Identifier for the service. To create custom UUIDs, you can use https://www.uuidgenerator.net/

Checkbox Fields

  • Read: Allows reading the characteristic (the client can read the characteristic value).
  • Write: Allows writing the characteristic (the client can modify the characteristic value). Then checked, a statement input with the 'On Write' event is added to the charactersitic to execute the corresponding code associated to the write action.
  • Notify: Allows to notify the client about the characteristic value.

Statement Instructions

  • On Write: When the write attribute is checked, this statement executes the corresponding code associated to the characteristic. Use the 'Characteristic Data' instruction to get access to the data (it's contents depends on the characteristic).

Output

Returns the characteristic.
