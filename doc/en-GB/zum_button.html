Read digital input (with pull-up resistor)

Instruction

Description

Returns HIGH or LOW depending on whether a digital input is connected or disconnected, respectively (typically on the buttons or microswitches). The input requires the use of a pull-up resistor for proper reading. This instruction activates the internal resistance of the Arduino itself (between 20K and 50K, depending on the microcontroller).

Inputs

  • Pin: Digital pin to read.

Output

Digital value read (HIGH or LOW).

Example

This examples shows how to use a pushbutton without an external resistor.
