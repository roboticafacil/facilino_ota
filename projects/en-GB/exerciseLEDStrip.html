Getting Started

Learn how set colors on a LED Strip.

Required material.
  • An Arduino or ESP board, i.e.: Arduino Uno, Arduino Nano, Wemos D1 R2 or Wemos D1 R32, NodeMCU, etc...
  • A LED Strip connected to D4 and a pushbutton connected to pins D2.

Basic Exercises
  1. Set a LED ON

    Set the first LED of a LED strip ON. Choose whatever color you want.

    Hint:

    Use the instruction to set the color of an individual LED (first instruction). Also set the correct number of LEDs based on the amount of connected LEDs in your LED strip.

  2. Set All LEDs ON

    Set ALL LEDs of a LED strip ON. Choose whatever color you want.

    Hint:

    Use the instruction to set the color of all LEDs (first instruction), combined with the second instruction 'All'. Also set the correct number of LEDs based on the amount of connected LEDs in your LED strip.

  3. Set LEDs Brightness I

    Set the brightness of LEDs by increasing its value from 0 to 100 in a for loop.

    Hint:

    In the 'Setup' section, set all LEDs off (black). Inside the 'Loop' section, use the instruction to set the brightness (fifth instruction) and after setting it value, set the desired color inside a for loop that varies between 0 and 100. Create a LOCAL variable to count the progress within the for loop.

  4. Set LEDs Brightness II

    Now, set the brightness of LEDs by increasing its value from 0 to 100 and then decreasing its value from 100 to 0.

    Hint:

    As in the previous exercise, use the same instructions, but now, implement a second for loop that counts from 100 to 0.

Intermediate Exercises
  1. All LEDs ON

    Set all LEDs ON when the pushbutton connected to pin D2 is pushed.

    Hint:

    Use the instruction that reads the button (sixth instruction) and set the LEDs on when the button is pressed.

  2. All LEDs ON and OFF

    Now, set all LEDs ON when the button is pushed and OFF when released.

    Hint:

    Based on the previous solution, clear all LEDs by setting its color to black when the button is released.

  3. All LEDs ON with Interrupt

    In the previous exercise, the button was read by polling, which means that the button must be pressed when the instruction is executed. In many cases, it is more convinient to use interrupts to do an action independently of the moment the button is pushed (the microcontroller will stop doing anything to attend the interrupt). So now, implement a similar behaviour as in the first intermediate exercise and set all LEDs ON when the button is pushed via an interrupt

    Hint:

    Use the seventh instruction to attend a falling edge interrupt (when the button is pushed). The button must be connected to D2 and the interrupt must be configured to use such pin.

  4. All LEDs ON and OFF with Interrupt

    Now, reimplement the second intermediate exercise so that the button is attached to an interrupt.

    Hint:

    Use the alternate case instruction to set ON or OFF the LEDs depending on the case to execute (the alternate instrucion will execute alternatively all cases introduced).
